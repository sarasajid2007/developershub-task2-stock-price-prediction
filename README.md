#  Task 2: Stock Price Prediction

This repository contains my submission for **Task 2: Stock Price Prediction** as part of the **DevelopersHub AI/ML Internship**.

##  Project Overview

The objective of this project is to build a Machine Learning model that predicts stock prices using historical stock market data. The project includes data preprocessing, feature engineering, model training, performance evaluation, and visualization of predicted versus actual stock prices.

---

##  Objectives

- Load and preprocess historical stock market data
- Perform Exploratory Data Analysis (EDA)
- Create relevant features for prediction
- Train a Machine Learning regression model
- Evaluate model performance
- Visualize actual vs predicted stock prices
